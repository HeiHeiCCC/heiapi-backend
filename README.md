# heiapi-backend
API平台后端主服务

# 项目介绍

基于React + SpringBoot + Dubbo + Gateway的 嘿API接口开放平台 。管理员可以接入并发布接口，可视化各接口的调用情况；
用户可以开通接口调用权限、浏览接口及在线调试，并可以通过客户端SDK轻松调用接口
